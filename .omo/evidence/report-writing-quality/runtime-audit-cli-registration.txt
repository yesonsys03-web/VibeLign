## uv run vib report --help
usage: vib report [-h] [--type TYPE] [--format {html,docx,pptx}]
                  [--output OUTPUT] [--force] [--date DATE] [--json]
                  [--polish] [--cli {auto,codex,opencode,agy,claude}]
                  [--emit-model] [--assist-missing] [--visual-cards]
                  [--reject-blocks REJECT_BLOCKS] [--polish-key POLISH_KEY]
                  [--theme THEME] [--title-font-size PT]
                  [--heading-font-size PT] [--body-font-size PT]
                  [--meta-font-size PT] [--heading-font FONT]
                  [--body-font FONT] [--author AUTHOR]
                  [--page-numbers | --no-page-numbers]
                  plan

기획안 마크다운을 업무/제안/결과 보고서 HTML 로 변환해요.

positional arguments:
  plan                  기획안 .md 경로

options:
  -h, --help            show this help message and exit
  --type TYPE           보고서 종류 (work=업무, proposal=제안, result=결과)
  --format {html,docx,pptx}
                        출력 포맷
  --output OUTPUT       저장 경로 (기본: .vibelign/reports/)
  --force               기존 --output 파일 덮어쓰기
  --date DATE           보고서 날짜 (기본: 오늘)
  --json                JSON 으로 결과 출력
  --polish              AI 로 어조 다듬기(무료 provider, 기본 OFF)
  --cli {auto,codex,opencode,agy,claude}
                        다듬기 provider (claude 는 명시 opt-in)
  --emit-model          다듬기 전/후 구조화 모델을 JSON 으로 출력(파일 미저장)
  --assist-missing      부족한 보고서 항목 보완 후보를 JSON 으로 출력(파일 미저장)
  --visual-cards        보고서용 카드뉴스 companion JSON 을 함께 출력
  --reject-blocks REJECT_BLOCKS
                        원본 유지할 블록 인덱스 JSON: [[section,block],...]
  --polish-key POLISH_KEY
                        emit 응답의 key — render 가 그 캐시 항목만 로드(재현성)
  --theme THEME         디자인 테마 (기본 classic)
  --title-font-size PT  타이틀 폰트 크기
  --heading-font-size PT
                        헤드라인 폰트 크기
  --body-font-size PT   본문 폰트 크기
  --meta-font-size PT   머리말(종류·날짜·작성자) 폰트 크기
  --heading-font FONT   제목 폰트 ID
  --body-font FONT      본문 폰트 ID
  --author AUTHOR       작성자 이름 (메타에 표시)
  --page-numbers, --no-page-numbers
                        페이지 번호(Word, 기본 ON)

이렇게 쓰세요:
  vib report plans/예약-앱.md --type work
  vib report plans/예약-앱.md --type proposal --json
