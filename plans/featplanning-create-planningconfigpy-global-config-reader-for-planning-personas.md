# 기획안 (보강본): 기획 도우미들의 "공통 설정 한곳 모으기"

## 한 줄 목표
기획방 도우미(페르소나)들이 **각자 따로 노는 대신, 한곳에 모아둔 공통 설정을 함께 읽어** 같은 기준으로 움직이게 합니다.

## 만들고 싶은 이유
지금은 기획방의 도우미들이 저마다 정해진 방식으로 동작해요. 그래서 "도우미가 쓰는 AI를 바꾸고 싶다"거나 "말투/언어를 통일하고 싶다" 같은 변경을 하려면 여러 군데를 따로 손봐야 할 수 있습니다.
설정을 **한 곳에 모아두면**, 한 번만 바꿔도 모든 도우미에게 똑같이 적용돼서 관리가 훨씬 쉬워집니다.
> ❓ 확인 필요: 만들고 싶은 진짜 이유가 "관리 편의"인가요, 아니면 "도우미마다 다른 AI/말투를 자유롭게 고르고 싶어서"인가요? (둘 다일 수도 있어요)

## 대상 사용자
- 기획방을 직접 쓰는 사람 중, **도우미별로 동작 방식(특히 어떤 AI를 쓸지)을 직접 고르고 싶은 사람**.
> ❓ 확인 필요: 이 설정을 만지는 사람이 "기획안을 쓰는 일반 사용자"인가요, 아니면 "기획방을 세팅해 두는 관리자/운영자"인가요? 둘은 필요한 화면이 달라져요.

## 핵심 기능
- 도우미들이 **공통으로 쓰는 설정을 한곳에서 읽어오기** (모든 도우미가 같은 설정을 바라보게)
- 도우미(페르소나)마다 **어떤 AI를 쓸지 지정**할 수 있기
- 설정이 비어 있거나 없을 때 쓸 **기본값** 정해두기
> ❓ 확인 필요: 여기서 정할 수 있는 항목이 "AI 선택"만인가요, 아니면 **언어·말투(존댓말/반말)·답변 길이** 같은 것도 포함하나요?

## 화면 또는 사용 흐름
- (예상 흐름) 설정을 **한 번 저장** → 기획방을 열면 도우미들이 저장된 설정을 읽어 **그대로 동작**.
> ❓ 확인 필요: 설정을 **바꾸는 화면(메뉴/UI)** 까지 이번에 만드나요? 아니면 이번엔 "읽어와서 적용되는 것"까지만 하고, 바꾸는 화면은 다음 단계로 미루나요?

## 제외할 것
- (제안) 설정을 편집하는 전용 화면은 이번 범위에서 빼고, 다음 단계로 둘 수 있어요.
> ❓ 확인 필요: 위 제안대로 "편집 화면 제외"가 맞나요? 빼고 싶은 다른 것이 또 있나요?

## 아직 결정이 필요한 질문
- 대상 사용자가 일반 사용자인가요, 운영자인가요?
- 공통 설정에서 고를 수 있는 항목 범위는 어디까지인가요? (AI만 / 언어·말투까지)
- 설정 편집 화면을 이번 범위에 포함하나요?
- 도우미마다 설정이 **서로 다를 때**와 **공통값을 따를 때**를 어떻게 구분할까요? (예: 개별 설정이 공통 설정보다 우선)

## 구현 전에 AI가 알아야 할 맥락
- 언어 설정: auto (자동) — 위에서 "언어를 공통 설정에 포함할지" 정해지면 이 칸도 같이 정리돼요.
> ❓ 확인 필요: 이미 비슷한 설정 방식(예: 다른 기능에서 쓰는 저장 위치/형식)이 있나요? 있으면 그것과 **모양을 맞추는 게** 좋아요.
