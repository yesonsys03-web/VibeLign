# VibeGuard: 코알못 & 바이브코딩 초보자를 위한 기능 제안 (vibe_skill_up.md)

VibeGuard의 핵심 철학인 **"AI 코딩 속도는 유지하되, 안정성과 구조적 품질을 지킨다"**를 바탕으로, 초보 사용자들이 더 안전하고 즐겁게 코딩할 수 있도록 돕는 아이디어 모음입니다.

---

## 1. 시각적 경험 개선 (Visual Experience) - 터미널 공포 제로화
'코알못'에게 검은 화면의 터미널은 마치 고장 난 기계를 대하는 것 같은 공포를 줍니다. 이를 해결하기 위해 VibeGuard를 **"눈에 보이고 만질 수 있는"** 도구로 진화시킵니다.

*   **VibeGuard 대시보드 (Local Web UI):** 터미널에 명령어를 치는 대신, 브라우저에서 버튼 클릭만으로 프로젝트를 관리합니다.
    *   **세이브 포인트 슬롯:** 마치 게임 저장 화면처럼, 체크포인트를 스크린샷(변경되거나 추가된 주요 파일의 핵심 코드 스냅샷)과 함께 카드 형태로 보여줍니다.
    *   **직관적인 'Undo' 버튼:** 빨간색의 큰 '되돌리기' 버튼을 배치하여, 언제든 안심하고 실험할 수 있는 심리적 안전망을 제공합니다.
*   **Vibe-Health 신호등:** 프로젝트의 상태를 복잡한 텍스트 대신 색깔로 표현합니다.
    *   🟢 **초록색:** "아주 안전해요! AI가 규칙을 잘 지키고 있어요."
    *   🟡 **노란색:** "잠깐, `main.py`가 조금 뚱뚱해졌어요. 나중에 정리하는 게 좋겠어요."
    *   🔴 **빨간색:** "위험! AI가 중요한 설정을 마음대로 바꿨어요. `Undo`를 추천합니다."
*   **비주얼 파일 맵 (Visual File Map):** 프로젝트의 파일 구조를 복잡한 폴더 트리가 아닌, **"마을 지도"**처럼 시각화합니다.
    *   중요한 파일은 큰 건물로, 임무를 수행 중인 AI는 캐릭터 아이콘으로 표시하여 "지금 AI가 어디서 무슨 일을 하고 있는지" 실시간으로 보여줍니다.
*   **AI 작업 캔버스 (Magic Canvas):** `patch` 명령어로 만들어진 요청서를 단순히 텍스트 파일이 아닌, **"예상 결과 이미지"**나 **"변경 지도"**로 보여줍니다. AI가 고칠 부분을 미리 하이라이트 해서 보여주므로, 코드를 읽지 못해도 어디가 바뀔지 알 수 있습니다.
*   **원클릭 설치 가이드 (Desktop App/Installer):** `uv install` 같은 명령어 대신, 일반 앱처럼 설치하고 실행할 수 있는 데스크톱 런처를 제공하여 터미널과의 첫 만남 자체를 제거합니다.

### [보충] 시각적 인터페이스의 3단계 구현 전략
터미널 공포를 단계적으로 없애기 위해 다음과 같은 방식으로 UI를 제공합니다.

1.  **1단계: Web UI (명령어 기반 자동 열기)**
    *   터미널에 `vibeguard dashboard` 한 줄만 치면 자동으로 크롬/사파리 브라우저가 열리며 관리 화면이 나타납니다.
2.  **2단계: Desktop App (독립형 앱)**
    *   카카오톡처럼 아이콘을 더블 클릭해 실행합니다. 터미널을 아예 켤 필요 없이 폴더 선택만으로 모든 기능을 제어합니다.
3.  **3단계: IDE Extension (코딩 도구 통합)**
    *   Cursor나 VS Code의 사이드바에 VibeGuard가 합쳐집니다. 코딩 도구 안에서 실시간으로 '신호등'을 확인하며 작업합니다.

## 2. 지능형 가이드 (Smart Assistant)
언제 어떤 명령어를 써야 할지 모르는 사용자에게 친절한 가이드를 제공합니다.

*   **VibeBuddy (대화형 가이드):** `vibeguard buddy` 모드를 통해 AI 코딩의 전 과정을 단계별로 안내받습니다. (예: "코드를 수정하기 전에 세이브 포인트를 만들까요?")
*   **One-Click 리팩토링:** `doctor`가 발견한 문제(예: 너무 큰 파일)를 버튼 하나로 자동 분리 및 정리해 주는 기능을 제안합니다.
*   **스마트 프롬프트 인핸서:** 사용자의 단순한 요청을 VibeGuard 규칙이 반영된 고품질 AI 프롬프트로 변환해 줍니다.

## 3. 교육적 피드백 (Educational Feedback)
성장을 돕는 친절한 설명을 덧붙입니다.

*   **"왜(Why)" 모드:** 경고나 제재가 발생했을 때, 프로그래밍 원칙을 초보자 눈높이에서 한글로 설명해 줍니다.
*   **튜토리얼 시나리오:** 특정 기능을 안전하게 추가하는 베스트 케이스 시나리오를 제공하여 실습을 돕습니다.

## 4. 실시간 보호 & 통합 (Safety & Integration)
AI의 돌발 행동으로부터 프로젝트를 더 견고하게 지킵니다.

*   **실시간 안심 알림 (Watch Notify):** `watch` 모드 중 AI가 보호된 파일을 건드리거나 규칙을 어기면 시스템 팝업 알림을 보냅니다.
*   **AI 도구 웹 연동:** ChatGPT/Claude 웹 UI에서 현재 프로젝트의 VibeGuard 상태를 프롬프트에 즉시 반영할 수 있는 도구를 제공합니다.
