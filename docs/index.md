# VibeLign Documentation

VibeLign (`vibelign`) is an AI coding safety CLI for vibe coding workflows.
It helps you protect project structure, save checkpoints, undo mistakes, manage anchors, and block secret leaks before commit.

## Start Here

- Install: `pip install vibelign`
- First setup: `vib start`
- Save before AI edits: `vib checkpoint "before work"`
- Undo bad changes: `vib undo`

## Why People Search for VibeLign

People usually look for VibeLign when they want:

- an AI coding safety CLI
- checkpoint and undo tools for AI edits
- anchor-based project structure protection
- secret protection before Git commit
- MCP support for Claude Code, Cursor, OpenCode, and Codex
