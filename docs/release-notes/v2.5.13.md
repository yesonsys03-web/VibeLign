## VibeLign v2.5.13 - Windows 정적 HTML 미리보기 로딩 수정

### 핵심 수정

- Windows `실행해보기` 미리보기 창에서 `index.html`이 빈 화면으로 멈추던 문제를 수정했다.
- 정적 HTML 미리보기를 `file://` 직접 로딩 대신 앱 내부 `vibelign-preview://localhost/...` 프로토콜로 제공하도록 변경했다.
- Windows WebView2가 내부 프로토콜을 `http://vibelign-preview.localhost/...` 형태로 다루는 경우도 허용했다.
- 정적 미리보기 중 `중지` 버튼을 누르면 실행 상태, 미리보기 URL, 정적 루트 상태를 즉시 해제하도록 보강했다.

### 검증

- `cargo test --manifest-path vibelign-gui/src-tauri/Cargo.toml commands::run_preview -- --nocapture`
- `cargo check --manifest-path vibelign-gui/src-tauri/Cargo.toml`
- `npm --prefix vibelign-gui run build`

### 테스트 안내

- Windows에서 `index.html`만 있는 프로젝트를 열고 `실행해보기`를 눌러 미리보기 창에 UI가 표시되는지 확인한다.
- 같은 상태에서 `중지` 버튼을 눌렀을 때 실행 상태가 즉시 해제되는지 확인한다.
