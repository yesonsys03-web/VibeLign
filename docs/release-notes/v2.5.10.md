## VibeLign v2.5.10 - Windows 실행해보기 흰 화면/종료 정리 수정

Windows에서 `실행해보기`를 누르면 미리보기 팝업이 흰 화면으로 멈추고, 앱 종료 후에도 프로세스가 남을 수 있던 문제를 수정했다. 정적 HTML 미리보기 주소를 WebView2에서 더 안정적인 IPv4 loopback 주소로 열고, 메인 창 종료 시 실행 미리보기 창과 하위 프로세스를 함께 정리한다.

### Fixed

- **Windows 정적 HTML 미리보기 흰 화면** - `localhost` 대신 `127.0.0.1`로 임베드 정적 서버 주소를 생성해 WebView2의 loopback 해석 차이로 인한 빈 화면 가능성을 줄였다.
- **앱 종료 후 프로세스 잔류** - 메인 창 닫기와 앱 종료 이벤트에서 실행 미리보기 창, watcher, 작업 실행, 실행 서버를 같은 흐름으로 정리한다.
- **임의 HTML 진입점 유지** - `index.html` 외 `quiz.html` 같은 감지된 HTML 진입점도 같은 IPv4 loopback URL로 열린다.

### Upgrade Notes

- Windows에서 v2.5.9 실행해보기 팝업이 흰 화면으로 멈추던 경우 v2.5.10 설치 후 같은 프로젝트의 `index.html` 실행을 다시 눌러 확인하면 된다.
- 앱을 닫을 때 남아 있던 실행 미리보기 서버나 하위 프로세스도 함께 정리되도록 보강했다.
