# VibeLign v2.5.1

Released: 2026-06-18

## English

VibeLign v2.5.1 adds a satgat-inspired report specimen pack and cleans up the temporary Claude warning UI.

- Added 13 satgat-inspired report specimen themes to the report theme catalog and GUI selector.
- Restored the planning room's default response priority to Chloe (Claude) first, then Gio (Codex).
- Removed the temporary `claude -p` credit warning from planning-room chips, persona settings, Work Room, and Design Preview after the policy reversal.
- Verified with focused GUI tests, TypeScript, ESLint on changed files, and the Vite production build.

## 한국어

VibeLign v2.5.1은 삿갓 스타일 보고서 specimen 팩을 추가하고, 임시 Claude 경고 UI를 정리한 릴리스입니다.

- 보고서 테마 catalog와 GUI 선택지에 삿갓 스타일 specimen 테마 13종을 추가했습니다.
- 기획방 기본 응답 우선순위를 다시 클로이(Claude) 먼저, 지오(Codex) 다음 순서로 복원했습니다.
- Claude 정책 변경이 취소되어 기획방 칩, 페르소나 설정, 작업방, 디자인 미리보기의 `claude -p` 크레딧 경고를 제거했습니다.
- GUI 집중 테스트, TypeScript, 변경 파일 ESLint, Vite production build로 확인했습니다.
